How do I install it?

- Download 'Clock in' and upzip the file.
- In your Chrome browser menu go to: Menu>Extentions
- Turn on 'developer mode' in the upper right corner
- Click "Load unpacked" in upper left corner
- Select the 'Clock in' folder
- Turn on extension (lever will turn blue)
- Open new tab and start timer
